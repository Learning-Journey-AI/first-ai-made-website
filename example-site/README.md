# example-site/

This folder is a **reference example only**. It is a copy of Nick Wagner's personal website, here to show Claude Code (and you) what a finished portfolio site looks like in practice.

## Two rules

1. **Do not edit this folder.** If you want to change something, change it in *your* site repo, not here.
2. **Do not deploy this folder as your own site.** It is not yours. It describes someone else.

## What it shows

- Plain HTML, CSS, and JavaScript with no build step.
- Pages: `index.html`, `projects.html`, `presentations.html`, `contact.html`.
- A simple `assets/` folder for CSS, JS, images, and the resume PDF.
- A `data/projects.json` file that drives the projects page from one editable list.

You don't need to read every file here. When you ask Claude Code to build your site, it will read this folder and use it as a model for structure and style.
