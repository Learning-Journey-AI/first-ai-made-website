# context-files/

Drop anything here that describes you. Claude Code will read everything in this folder when it builds your website.

## What to put here

- **Your resume.** PDF, Word, plain text — any format is fine.
- **A photo of yourself.** Headshot or any photo you'd be happy to put on a website. JPG or PNG.
- **(Optional) A short bio**, if you want one. A `.txt` or `.md` file is great. Otherwise Claude will write one from your resume.
- **(Optional) Project notes.** A list of projects you want to show off. One per paragraph, with what it was, what you did, and a link if you have one.
- **(Optional) Links.** LinkedIn, GitHub, personal email, social profiles — drop them in a `links.txt` if you like.
- **(Optional) Anything else relevant.** Writing samples, certificates, talks you've given, awards. If it belongs on the site, put it here.

## Rules of thumb

- File names and formats don't matter. Claude Code will figure out what each file is.
- The more you put here, the better the site can be. But a resume + a photo is enough to start.
- Claude Code will not change or delete anything in this folder without asking you first.
